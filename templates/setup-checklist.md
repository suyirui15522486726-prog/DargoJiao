# DargoJiao 首次验收清单

## 环境

- [ ] Codex Desktop 已登录并能访问本地项目。
- [ ] `./scripts/doctor.sh` 没有 `FAIL`。
- [ ] 飞书当前用户授权有效。
- [ ] 当前用户能查看输入群历史消息。
- [ ] 当前用户能读取并编辑目标知识库文档。

## 自动化

- [ ] 只有一个 DargoJiao 计划任务。
- [ ] 工作日、执行时间、时区和回看天数符合预期。
- [ ] 任务关联的是正确本地项目和 Git ref。
- [ ] Codex「已安排」中能看到手动运行结果。

## 笔记

- [ ] 测试链接进入正确 Topic 分类。
- [ ] 笔记包含标题、总结时间、Topic、错误答题和高分答题。
- [ ] 项目无关时省略项目段；相关时包含相对路径、符号和代码块。
- [ ] 原视频链接可以点击。
- [ ] 去重信息包含消息键和视频键。

## 去重与恢复

- [ ] 相同任务连续运行两次，笔记仍只有一份。
- [ ] 同一视频再次转发时被跳过。
- [ ] 证据不足的链接标为待重试且不占序号。
- [ ] 恢复权限或网络后，待重试链接可以再次处理。

## 运行结果

- [ ] 成功报告包含成功数、跳过数、待重试数和分类。
- [ ] 异常报告包含时间、阶段、原因和用户动作。
- [ ] 成功和异常结果都能在 Codex「已安排」中查看。
