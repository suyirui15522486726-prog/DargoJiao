# 去重与重试契约

## 三层键

1. 主键：`svnote-msg:<message_id>`，保证同一飞书消息最多成功处理一次。
2. 视频键：规范化 URL 和平台视频 ID，保证同一视频被不同消息重复转发时不重复生成。
3. 标题：只作冲突提醒；同名内容可能不是同一视频，不能单独决定跳过。

## 处理前

在目标知识库的候选文档中依次查找主键、视频 ID 和规范化 URL。任一强唯一键已存在时跳过，并记录命中的键。

## 写入

1. 先确定目标分类和下一连续序号。
2. 再执行一次处理前查询，避免并发运行导致重复写入。
3. 把主键、规范化 URL 和平台视频 ID 与正文一起写入。
4. 写入后回读验证：主键必须恰好出现一次，标题对应的编号标题也必须恰好出现一次。

## 失败与重试

- 证据不足、网络失败、权限失败或文档写入失败时，不写任何成功标识，也不占用序号。
- 外部写入成功但回读失败时，下一次先查询唯一键；命中后视为已完成，不再次追加。
- 单条失败不阻止同批其他链接。
- 超出回看窗口的消息不会自动补处理；排障时可临时扩大窗口后手动补跑。
